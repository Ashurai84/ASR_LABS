
### 2026-02-09 • DECISION
**Shifted AI Inference to the Edge (Browser)**
Navigating urban environments requires split-second hazard alerts. Relying on a backend API for computer vision would introduce dangerous latency (network roundtrips). We decided to run the coco-ssd model directly on the user's device using TensorFlow.js in the browser.

> Outcome: Achieved near-zero latency for object detection and auditory warnings, keeping users safe offline or on poor network connections.
